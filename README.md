### SubEnoom

Sub domain enumerator that can intake domains, IPs, and subnets. 

##### Syntax

```
./SubEnum.sh ./domains.txt outputdirname ( all|passive )
```
```
-c  tool cero will only return results with same domain(s) that are submitted to the tool
```



#### Domain file format:

```
example.com
example2.com
1.1.1.1
```
